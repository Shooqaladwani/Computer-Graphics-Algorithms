# 🖥️ University Campus Graphics

A Computer Graphics project developed in C++ using the WinBGIm graphics library (graphics.h). The project demonstrates drawing and animation techniques by creating a simple university campus scene.

---

## Features

- University building
- Road and parking area
- Sun
- Student character
- Bus drawing
- Static version
- Animated version with moving bus
- Double buffering animation

---

## Technologies Used

- C++
- Visual Studio 2022
- WinBGIm (graphics.h)
- Windows API
